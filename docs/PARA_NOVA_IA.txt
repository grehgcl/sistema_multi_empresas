🔧 COMANDOS ÚTEIS
=========================================

Iniciar servidor (modo local - LOG)
npm start

Iniciar com WhatsApp Evolution (envia mensagens reais!)
node -r dotenv/config server.js dotenv_config_path=.env.local

Migração do banco (dias_bloqueio individual)
node scripts/migrate-dias-bloqueio.js

Migração do banco (dias_bloqueio_geral)
ALTER TABLE empresas ADD COLUMN IF NOT EXISTS dias_bloqueio_geral INTEGER DEFAULT 0;

Adicionar telefone do dono e endereço
ALTER TABLE empresas ADD COLUMN telefone_dono VARCHAR(20);
ALTER TABLE empresas ADD COLUMN endereco TEXT;

Testar job de lembretes manualmente
node -e "require('./server/jobs/lembretes.js').enviarLembretes()"

Ver logs do servidor
npm start

Acessar sistema
http://localhost:3000
